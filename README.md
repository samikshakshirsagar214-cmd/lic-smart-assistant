LIC Smart Assistant

A Python-based smart assistant for LIC (Life Insurance Corporation) that helps interact with LIC policy data and perform useful tasks related to LIC policy documents, including policy lookup and intelligent assistance.

This project includes:

A Python application (app.py) that acts as the core assistant.

LIC policy documents used as data sources.

Required dependencies listed in requirements.txt.
