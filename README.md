# LIC Smart Assistant (GenAI + RAG)

## Overview
An intelligent Retrieval-Augmented Generation (RAG) based assistant designed to answer queries from LIC insurance documents using Large Language Models.

## Tech Stack
- Python
- LangChain
- Large Language Models (LLMs)
- Vector Database (FAISS / Chroma)
- Streamlit

## Features
- PDF document ingestion
- Embeddings generation
- Context-aware question answering
- Interactive Streamlit UI

## Architecture
PDFs → Text Chunking → Embeddings → Vector DB → LLM → Answer

## How to Run
1. Clone the repository
2. Install requirements
3. Run Streamlit app

## Use Case
Helps users quickly extract accurate information from insurance documents.
